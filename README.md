# NEON RACE

NEON RACE は、Three.js と Vite で作成したネオンサイバーパンク風の3Dレーシングゲームです。

3レーンの道路を走りながら、コインを集め、障害物を避け、ブーストアイテムを取得してスコアを伸ばします。

## 起動方法

初回のみ依存関係をインストールします。

```
npm install
```

開発サーバーを起動します。

```
npm run dev
```

キャッシュを無視して起動したい場合は、次を使用します。

```
npm run dev -- --force
```

ビルド確認をする場合は、次を使用します。

```
npm run build
```

## 操作方法

| キー           | 操作                     |
| -------------- | ------------------------ |
| `SPACE`      | ゲームスタート           |
| `A` / `←` | 左レーンへ移動           |
| `D` / `→` | 右レーンへ移動           |
| `W` / `↑` | 加速                     |
| `S` / `↓` | 減速                     |
| `P`          | ポーズ / 再開            |
| `R`          | リスタート               |
| `Shift + R`  | 状況に応じたリセット操作 |

## ゲーム内容

* 3レーン制の3Dレーシングゲーム
* コイン取得でスコア加算
* ブーストアイテム取得で一時的に加速
* 障害物に衝突するとゲームオーバー
* 距離、スコア、コイン数、ミッションを表示
* ハイスコアはブラウザの localStorage に保存

## スコア

| 内容                   | スコア                     |
| ---------------------- | -------------------------- |
| コイン取得             | +10                        |
| ブースト中のコイン取得 | +20                        |
| ミッションクリア       | ミッション内容に応じて加算 |

## ミッション

プレイ中に複数のミッションが出題されます。

例：

* コインを一定数集める
* 一定時間生き残る
* ブーストを複数回取得する
* 一定スコアに到達する

ミッションをクリアすると、追加スコアが加算されます。

## 車両モデル

車両モデルは以下の場所に配置します。

```
public/models/car.glb
```

`<span>car.glb</span>` が存在する場合はGLBモデルを読み込みます。
存在しない場合は、フォールバック用の簡易車両が表示されます。

## 500 kB warning について

`<span>npm run build</span>` 実行時に、以下の警告が出る場合があります。

```
Some chunks are larger than 500 kB after minification
```

これは、ビルド後のJavaScriptファイルが500kBを超えているという警告です。
現時点ではビルド失敗ではないため、エラーではありません。

将来的に本番公開や表示速度の最適化を行う場合は、コード分割やファイル軽量化を検討します。
