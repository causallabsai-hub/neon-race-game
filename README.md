# NEON RACE

ネオンサイバーパンク風の3Dレースゲームです。Three.js製。

## 操作方法

| キー                     | 動作                                      |
| ------------------------ | ----------------------------------------- |
| `SPACE` / 画面クリック | ゲームスタート                            |
| `A` / `←`           | 左レーンへ移動                            |
| `D` / `→`           | 右レーンへ移動                            |
| `W` / `↑`           | 加速                                      |
| `S` / `↓`           | 減速                                      |
| `P`                    | ポーズ / 再開                             |
| `R`                    | リスタート（ゲームオーバー時 / ポーズ中） |

### スマホ操作

画面下部の半透明タッチボタンで操作できます（タッチデバイスで自動表示）。

## ゲーム仕様

### 基本ルール

- 3レーン（左・中央・右）の道路を走行
- 赤・オレンジの障害物に当たるとゲームオーバー（ブースト状態・速度状態も即終了）
- コイン（黄色のリング）を集めてスコアを稼ぐ
- ブーストアイテム（青い八面体）を取ると3秒間速度アップ＆コイン得点2倍

### スコア

| 状況                     | 得点 |
| ------------------------ | ---- |
| コイン取得（通常）       | 10点 |
| コイン取得（ブースト中） | 20点 |
| ミッションクリア         | ボーナスポイント（ミッションごとに異なる） |

### ミッション

1プレイ中に4つのミッションが順番に出題されます。クリアするとボーナスポイントが加算され、通知が表示されます。同じミッションで重複して報酬は入りません。全ミッションクリアで追加ボーナスあり。

| ミッション           | ボーナス |
| -------------------- | -------- |
| コインを10枚集める   | +100     |
| 30秒生き残る         | +100     |
| ブーストを2回取る    | +150     |
| スコア1000点達成     | +200     |
| 全ミッションクリア   | +500     |

### 難易度

- 最初30秒: やさしい（障害物スポーン間隔が長い）
- 30〜120秒: 徐々に難化
- それ以降: 最高難易度固定

### 評価ランク

| スコア  | ランク |
| ------- | ------ |
| 800以上 | S      |
| 400以上 | A      |
| 150以上 | B      |
| 149以下 | C      |

## 機能一覧

- **車のカラー選択**: スタート画面で赤・シアン・黄色・白から選択（localStorage保存）
- **GLB車両モデル**: `public/models/car.glb` を自動読み込み。ファイルが存在する場合はGLBモデルを表示。存在しない場合は簡易フォールバック車両を表示
- **ミッション**: 固定4ミッションを順番にクリア。達成時に画面中央へネオン通知表示＆ボーナスポイント加算
- **景色テーマ**: 走行距離に応じて4種類の夜景テーマ（星空・ブルー・パープル・グリーン）に自動切替
- **道路カーブ演出**: 道路にゆるいカーブ感があり、カメラも追従
- **ポーズ**: Pキーで一時停止／再開
- **レーンHUD**: 右下に3レーンの簡易表示。障害物（赤）・コイン（黄）・ブースト（青）を可視化
- **ゲームオーバー画面**: スコア・距離・コイン数・最大速度・評価ランク・ミッション結果を表示
- **ハイスコア**: localStorage に自動保存

## 車両モデルのカスタマイズ

`public/models/car.glb` にGLB形式の車両モデルを配置するとゲーム内で使用されます。

- ファイル名は必ず `car.glb` にする
- 無料モデルを使う場合はライセンス・商用利用可否を確認する
- モデルのサイズや向きが合わない場合は `src/main.js` 内の以下の定数を調整する

```js
const GLB_CAR_SCALE      = 1.0;   // 大きさ
const GLB_CAR_ROTATION_Y = Math.PI; // 向き（ラジアン）
const GLB_CAR_POSITION_Y = 0.0;   // 高さ
```

## 技術スタック

- [Three.js](https://threejs.org/) — 3Dレンダリング（GLTFLoader含む）
- [Vite](https://vitejs.dev/) — ビルドツール
- Web Audio API — エンジン音・効果音

## 起動方法

```bash
npm install
npm run dev
```

## 開発時の注意

- APIキー・シークレットはコードに直接書かない（`.env` で管理し `.gitignore` に追加する）
- 有料API・有料素材を使う場合は事前確認する
- 外部モデルや素材を使う場合はライセンスを確認する
- Console に赤いエラーが出ないことを確認する
- 変更後は `npm run build` でエラーがないことを確認する
